POLITIQUE DE CONFIDENTIALITÉ

Dernière mise à jour : 23 février 2026

En résumé : Nous collectons le minimum de données nécessaires au fonctionnement de l'Application. Vos données d'habitudes vous appartiennent. Nous ne vendons rien à personne.

1. Responsable du traitement

PROJET FEVRIER est édité par un développeur indépendant basé à Lyon, France.

2. Données collectées

Données de compte :

  • Adresse email (pour l'authentification)
  • Pseudo et avatar (choisis par l'utilisateur)

Données d'utilisation :

  • Habitudes créées et leur suivi quotidien
  • Statistiques de progression (scores, streaks, XP)
  • Messages dans les chats de groupe
  • Participation aux challenges

Données techniques :

  • Données de connexion Firebase (horodatage)

3. Finalité du traitement

Les données sont utilisées exclusivement pour :

  • Permettre le fonctionnement de l'Application
  • Afficher vos statistiques et votre progression
  • Permettre les interactions sociales (groupes, chat, leaderboard)

4. Stockage des données

Données locales : vos habitudes et statistiques sont stockées dans le localStorage de votre navigateur.

Données cloud : les données de compte, groupes et messages sont stockées sur Firebase (Google Cloud Platform), dont les serveurs sont situés dans l'Union Européenne.

5. Partage des données

Vos données ne sont jamais vendues ni partagées avec des tiers à des fins commerciales.

Seules les données suivantes sont visibles par d'autres utilisateurs dans les groupes :

  • Pseudo et avatar
  • Score du jour et statistiques dans le leaderboard
  • Messages envoyés dans les chats de groupe

6. Durée de conservation

Vos données sont conservées tant que votre compte est actif. La suppression du compte entraîne la suppression de vos données personnelles.

7. Vos droits (RGPD)

Conformément au Règlement Général sur la Protection des Données, vous disposez des droits suivants :

  • Accès : consulter vos données via l'Application
  • Rectification : modifier vos informations (pseudo, bio)
  • Suppression : supprimer votre compte et toutes vos données via les paramètres
  • Portabilité : exporter vos données en CSV via les paramètres
  • Opposition : vous pouvez cesser d'utiliser l'Application à tout moment

8. Cookies

L'Application n'utilise pas de cookies tiers. Seul le localStorage du navigateur est utilisé pour stocker vos préférences et données locales.

9. Sécurité

L'authentification est gérée par Firebase Authentication (Google). Les communications sont chiffrées via HTTPS. Les règles de sécurité Firestore contrôlent l'accès aux données.

10. Modifications

Cette politique peut être mise à jour. La date de dernière modification est indiquée en haut du document.

11. Contact

Pour exercer vos droits ou poser une question, contactez-nous via l'Application.
